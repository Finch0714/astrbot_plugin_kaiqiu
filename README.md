# astrbot_plugin_kaiqiu — 开球网数据查询插件

查询开球网球员与赛事数据。

## 指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/kqw player <名称>` | 查询球员信息 | `/kqw player 球员名称` |
| `/kqw match <赛事>` | 查询比赛信息 | `/kqw match 比赛名称` |

## 安装

将本插件目录放入 `AstrBot/data/plugins/`，在 WebUI 中启用即可。

## 配置

暂无。后续按需扩展。
